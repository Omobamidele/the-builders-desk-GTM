# Cold Email

Copy, sequences, deliverability guides, and scripts for cold outbound that doesn't land in spam.

- `copy/` — subject lines, individual emails, frameworks
- `sequences/` — full multi-touch sequences with reasoning for each step
- `deliverability/` — domain warmup, DNS setup, spam-trigger checklists
- `scripts/` — code for sending, personalization, list hygiene
