# Prompting

A prompt library organized by GTM use case, not just "good prompts" in the abstract.

- `by-use-case/` — prompts organized by job to be done (cold email personalization, lead scoring, call summarization, etc.)
- `patterns/` — reusable prompting patterns/structures you can adapt to any use case
